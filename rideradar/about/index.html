About RideRadar

RideRadar is a lightweight search project by Torrin Kay (torkay.com) that helps Australians browse vehicle listings faster. We aggregate public listing metadata and present it in a clear, comparable format to save shoppers time.

What we do

  • Fetch public listing data from multiple sources (official APIs where available, and compliant HTML fetching elsewhere).
  • Normalize specs such as price, odometer, body, drivetrain, and photos for easier comparison.
  • Link you to the original listing to enquire or purchase directly with the seller/marketplace.

Data & privacy

RideRadar does not process personal user data. We handle only public listing information and basic, privacy-respecting operational logs. If analytics are enabled, they are aggregate and non-identifying.

For details, see our Privacy Policy.
